Simulador de Substituição de Páginas: FIFO vs LRU
Descrição
Este projeto implementa e compara dois algoritmos clássicos de substituição de páginas em sistemas de memória virtual:

FIFO (First In First Out): Remove a página mais antiga na memória
LRU (Least Recently Used): Remove a página menos recentemente utilizada

O simulador permite visualizar o comportamento de cada algoritmo, comparar seu desempenho e entender suas características através de análise crítica detalhada.
